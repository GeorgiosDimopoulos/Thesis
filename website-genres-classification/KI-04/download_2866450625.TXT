Enigmail for Thunderbird is available from the Enigmail for Thunderbird page. 

Standard Download

Enigmail is a "plugin" that works with the mail software that is part of Mozilla, Netscape 7.x or Beonex 0.8.x. Enigmail is dually-licensed under the GNU General Public License and the Mozilla Public License . 

To install Enigmail, load this page using Mozilla (or Netscape 7.x or Beonex 0.8.x). Then click on one of the links below, depending upon your Mozilla/Netscape/Beonex version, to initiate the download and install. After installation, refer to the Help page for more information. If you are using a localized version of Mozilla, check the Language Packs page to see if Enigmail is available in your language. 

Enigmail downloads are available for the following Mozilla/Netscape/Beonex versions (please read the Notes below before installing): 

1.  For Mozilla 1.6 on Windows, Linux/x86, Mac OS X, FreeBSD, Solaris 8 

2.  For Mozilla 1.5/1.5.1 on Windows, Linux/x86, Mac OS X, FreeBSD, NetBSD, Solaris 8 

3.  For Mozilla 1.4/1.4.1 or Netscape 7.1 on Windows, Linux/x86, Mac OS X, FreeBSD, Solaris 8 

4.  For Mozilla 1.3.1/1.3 on Windows, Linux/x86, Mac OS X, FreeBSD 

5.  For Mozilla 1.2.1/1.2 on Windows, Linux/x86 

6.  For Mozilla 1.1 on Windows, Linux/x86, FreeBSD 

7.  For Mozilla 1.0.2, Netscape 7.02/7.01 or Beonex 0.8.2 on Windows, Linux/x86 

8.  For Mozilla 1.0/1.0.1, Netscape 7.0 or Beonex 0.8.1 on Windows, Linux/x86, Solaris8/Sparc 

9.  For Mozilla 1.0 on Linux/PPC and FreeBSD or Netscape7 Preview Release1 on Windows/Linux (older version) 

As of Enigmail 0.74.0 (Mozilla 1.3), the versions for Mac OS X are available on this page. For older versions, go to the Test Download page. 

Notes: 

1.  Enigmail has only been tested with the milestone releases of Mozilla or Netscape. If you use a nightly build, a third-party build, or your own build of Mozilla, Enigmail may not always work and may even crash Mozilla! 
2.  To use Enigmail, you also need to have GPG (Gnu Privacy Guard) installed on your system. PGP is no longer supported. See the Help page for more information on using Enigmail. 
3.  The Install button also installs the Enigmime module, which is required for inter-process communication with GPG. The Enigmime binaries are provided only for selected platforms, including Windows and Linux (x86). If you wish to use Enigmail on other platforms, you will need to compile Enigmime yourself. The Enigmime module should compile without modifications on most Unix-like platforms. 
4.  On many platforms, Enigmail will only work properly if compiled with the same compiler version as Mozilla was compiled (currently gcc 3.2 for Linux). If you are unsure, or receive some error messge, then read the Installation FAQ! 
5.  If you wish to download the XPI file(s) without installing, use the hyperlinks instead of the Install button. (You will need to shift-click or use the context menu to avoid the installation dialog which pops up if you simply click on the hyperlinks).

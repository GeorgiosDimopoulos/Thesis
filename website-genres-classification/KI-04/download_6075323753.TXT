Download Ruby

---------------------------------------------------------------------

How to get Ruby.

* The stable version ruby-1.6.8. 
* The CVS branch for the development version is available. See Ruby CVS Repository Guide. 

Downloadable Items

* Ruby version 1.8.0-preview1 source (development release) and an errata patch 

* Ruby version 1.6.8 source (stable release) 

* Ruby 1.4.6 Reference Manual 
* You can get Ruby-mswin32. 

* Reference Manual for 1.6.8 is not yet available. Buy your copy of "Programming Ruby", or get online free version. 

* Stable snapshot is available. This is tar'ed and gzip'ed file of the latest stable CVS. It should be better than the last stable release. 

* Nightly snapshot is available. This is tar'ed and gzip'ed file of the latest CVS. It may contain unfixed problems. (as usual ;-) 

* Install Ruby under Windows ... A single download that contains everything you need to run Ruby under various Windows operating systems. 

* Windows: Ruby for Windows users - Includes easy starter kit for Microsoft Windows users. 

* Ruby on Windows
